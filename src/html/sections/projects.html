Proyectos

En este proyecto tiene lugar una aplicación web basada en React llamada Jmmming, la cual utiliza la API de Spotify para permitir a los usuarios buscar sus propias listas de reproducción, crear una nueva lista, y añadir nuevas canciones a sus ya existentes listas de reproducción.

In this project, I have built a React web application called Jammming which uses the Spotify API to allow users to search their own Spotify library, create a custom playlist, save it to their Spotify account or add new songs to already existing playlists.
